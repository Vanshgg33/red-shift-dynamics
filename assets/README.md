# Assets Folder

This folder contains all static assets (images, icons, etc.) for the project.

## Usage

Import assets using the `@assets` alias:

```tsx
// Images
import logo from '@assets/logo.png';
import heroImage from '@assets/heroBackdrop.avif';

// Usage in components
<img src={logo} alt="Logo" />
<img src={heroImage} alt="Hero" />
```

## Available Assets

- `logo.png` - Company logo
- `heroBackdrop.avif` - Hero section background
- `footerBackdrop.avif` - Footer background
- `mainBg.png` - Main background image
- `image.png` - General purpose image
- `favicon.ico` - Favicon
